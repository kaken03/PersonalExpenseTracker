# Personal Expense Tracker

A full-stack Django web application for tracking personal expenses with user authentication, CRUD operations, and deployment-ready configuration.

Features

- **User Authentication**: Registration, login, and logout functionality
- **Expense Management**: Create, read, update, and delete expenses
- **Categorization**: Organize expenses by category (Food, Transport, Bills, Shopping, Other)
- **Dashboard**: View monthly totals and recent expenses at a glance
- **Filtering & Search**: Filter expenses by category and search by description
- **Monthly Analytics**: Automatic calculation of monthly expense totals
- **Responsive Design**: Bootstrap 5 for mobile-friendly interface
- **Database Flexibility**: SQLite for development, PostgreSQL for production
- **Deployment Ready**: Configuration for Railway deployment with environment variables
